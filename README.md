# Honeycomb
A website developed in ASP.NET, featuring authentication and authorization, as well as security measures.

## Features:
 - Sick aesthetic.
 - Authorization measures for two distinct user classes: user and admin.
 - Authentication measures.
 - Validation measures.
 - It's got *soul.*
